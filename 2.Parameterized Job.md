### Project: Jenkins Parameterized Job

#### Job Creation:
Create a new item (Freestyle project) named parameterized-job.
- Parameters: `String Parameter '`
  Name: Stage
  Default Value: Build

- Choice Parameter:
  Name: env
  Choices (one per line): Development, Staging, Production

- Build Execution:
  Add a build step: Execute shell.
  Command to add:
```Bash
echo "Stage: $Stage"
echo "Environment: $env"
Verification:
```
- Run Build with Parameters.

- Select Staging for the env parameter and click Build.

- Verify the build passes successfully in the console output.
